# Creating a Golang project.

1. Create directory structure.

```text
.docs/
.github/
  .workflows/
    ci.yml
    release.yml
cmd/
  skunk_app/
internal/
.gitignore

```

2. Create Module

```bash
go mod init github.com/estenrye/skunk
```
